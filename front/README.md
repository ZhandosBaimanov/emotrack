# Emotrack Frontend

Фронтенд приложения для отслеживания эмоционального состояния.

## Технологии

- **React 18** - UI библиотека
- **Vite** - сборщик
- **Tailwind CSS** - стилизация (Glassmorphism дизайн)
- **React Router** - маршрутизация
- **Axios** - HTTP клиент
- **Lucide React** - иконки

## Структура проекта

```
front/
├── public/              # Статические файлы
├── src/
│   ├── api/            # API клиент с interceptors
│   │   └── api.js
│   ├── components/     # Общие компоненты
│   │   └── ProtectedRoute.jsx
│   ├── context/        # React контексты
│   │   └── AuthContext.jsx
│   ├── pages/          # Страницы
│   │   ├── Login.jsx
│   │   ├── Register.jsx
│   │   ├── UserDashboard.jsx
│   │   └── PsychologistDashboard.jsx
│   ├── App.jsx         # Главный компонент с маршрутизацией
│   ├── main.jsx        # Точка входа
│   └── index.css       # Глобальные стили
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.js
```

## Установка

```bash
cd front
npm install
```

## Запуск

```bash
# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Превью продакшен сборки
npm run preview
```

## Функционал

### Страница регистрации

- Поля: Email, Username, Password
- Переключатель роли: "Я пациент" / "Я психолог"
- Условное поле "Код психолога" для пациентов
- Сохранение JWT токена в localStorage

### Дашборд пациента

- Приветствие и кнопка выхода
- Выбор настроения (5 карточек с эмодзи)
- Поле для комментария
- История эмоций

### Дашборд психолога

- Уникальный код с кнопкой копирования
- Список привязанных пациентов
- Просмотр истории эмоций пациентов
- График настроения и статистика

### Защита маршрутов

- Автоматическое перенаправление на логин без токена
- Разграничение доступа по ролям
- Автоматическое добавление токена к запросам

## API

Приложение проксирует запросы на `http://localhost:8000` через Vite прокси:

- `/api/*` → `http://localhost:8000/*`

## Дизайн

- Glassmorphism стиль для карточек
- Спокойная цветовая палитра (зеленый, синий, серый)
- Адаптивный дизайн для мобильных устройств
